用于市场数据更新的公共频道（Level 2 价格数据）。使用 asset ID 订阅以接收订单簿快照、价格变化、交易执行和市场事件。

端点
wss://ws-subscriptions-clob.polymarket.com/ws/market

订阅
{
"assets_ids"
: [
"<token_id_1>"
,
"<token_id_2>"
],
"type"
:
"market"
,
"custom_feature_enabled"
:
true
}
设置
custom_feature_enabled: true
以接收
best_bid_ask
、
new_market
和
market_resolved
事件。
