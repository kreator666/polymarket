Polymarket 是一个预测市场平台，用户可以在这里就现实世界事件的结果进行交易。与传统博彩不同，你在一个开放的点对点市场中直接与其他用户交易。价格反映了市场对某一事件发生概率的集体判断。
平台采用非托管模式，你始终掌控自己的资金。所有交易通过区块链上的智能合约结算，确保操作透明且无需信任第三方。

Polymarket 运作机制

价格即概率
Polymarket 上每个份额的价格在
$0.00
到
$1.00
之间。价格代表市场对该结果发生概率的判断。
例如，如果某事件的 “Yes” 份额交易价格为
$0.65
，意味着市场认为该事件发生的概率约为
65%
。

抵押品与代币
Polymarket 使用 USDC.e（Polygon 上的桥接 USDC）作为抵押品。每一对 Yes/No 份额都有完全的资金支撑：
$1 USDC.e
可铸造一份 Yes 份额和一份 No 份额
获胜份额可兑换
$1.00
失败份额价值
$0.00
份额以代币形式表示，基于
Gnosis Conditional Token Framework
（ERC1155 标准），实现无缝的链上交易与结算。

交易
Polymarket 使用点对点订单簿（CLOB）进行交易。你直接与其他用户交易，而非与庄家对赌。
买入份额
— 当你认为市场低估了某一结果的概率
卖出份额
— 当你认为市场高估了某一结果的概率
随时退出
— 在判定结果公布前卖出持仓，锁定利润或止损
操作
适用场景
盈利条件

事件发生

判定
当事件结束后，市场通过
UMA Optimistic Oracle
进行判定：
提议者提交结果并缴纳保证金
进入争议期，任何人都可以提出异议
如有争议，UMA 代币持有者投票决定正确的判定结果
获胜代币可兑换 $1 USDC.e
这一社区驱动的流程确保了市场判定的公正性和准确性。

为什么选择区块链
Polymarket 构建在
Polygon
区块链网络上，主要基于以下原因：
全球可访问
— 任何有互联网连接的人都可以参与
非托管
— 资金由你掌控，而非中心化机构
透明
— 所有活动均可在链上公开验证
快速且低成本
— Polygon 提供快速、低费用的交易体验
价值稳定
— USDC.e 与美元 1:1 锚定，避免加密货币的价格波动

代理钱包
当用户首次在 Polymarket.com 上进行交易时，系统会提示创建钱包。此时会在 Polygon 上部署一个 1-of-1 多签合约，由用户的 EOA（MetaMask 钱包或 MagicLink 钱包）控制和拥有。这个代理钱包存放用户的所有持仓（ERC1155）和 USDC.e（ERC20）。
使用代理钱包使 Polymarket 能够提供更好的用户体验——多步骤交易可以原子化执行，交易也可以通过 Gas Station Network 的 relayer 来中继。如果你是开发者，想要以编程方式访问通过 Polymarket.com 界面积累的持仓，你可以继续使用智能合约钱包（通过所有者账户执行交易），也可以使用所有者账户将资产转移到新地址。
