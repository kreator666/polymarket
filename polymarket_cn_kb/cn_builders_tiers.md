Unverified
100 transactions/day
所有新 builders 的默认等级。无需审批即可立即开始。
如何开始:
前往
polymarket.com/settings?tab=builder
创建 builder 个人资料
点击
”+ Create New”
生成 API keys
实现
builder signing
— 访问 Relayer 和 CLOB 订单归因所必需
包含内容:
通过 Safe/Proxy 钱包在所有 CLOB 订单上进行无 gas 交易
通过 Safe/Proxy 钱包在每日限制内的所有 Relayer 交易上补贴 gas
订单归因到你的 builder 个人资料
访问所有客户端库和文档

如何升级
1
构建和发布
从 Unverified 等级开始构建你的集成。
2
产生交易量
通过 Polymarket 路由订单并展示持续的使用情况。
3
申请验证
发送邮件至
builder@polymarket.com
,附上你的
builder key 和用例。
4
获得批准
Polymarket 团队会审核申请并在几个工作日内回复。

联系方式
准备好升级或有疑问?
builder@polymarket.com
发送邮件给我们,附上你的 Builder API Key 和用例详情。

FAQ
如何知道我是否已验证
验证状态会显示在你的
Builder Profile
设置中。
如果我超出每日限制会怎样
超出每日限制的 Relayer 请求会受到速率限制并返回错误。如果你经常达到限制,考虑升级到
Verified 或 Partner 等级。
我能获得临时限制提升吗
对于特殊活动或产品发布,请联系
builder@polymarket.com
。
